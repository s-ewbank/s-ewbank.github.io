MARIPoSA
[Download] [Documentation]
MARIPoSA (Manageable And Reproducible Integrated Pose Segmentation Analysis) is a GUI and Python command line-based platform for unified analysis of naturalistic animal and human pose segmentation and estimation data in the native formats of a variety of open source tools.
